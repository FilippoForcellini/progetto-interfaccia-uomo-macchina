# Progetto Interfaccia Uomo-Macchina

## Introduzione

Questo progetto prevede la gestione dei turni di lavoro dei dipendenti di un porto. L'utente potrà gestire i dipendenti, creare nuove navi e assegnare per ogni nave un tot di dipendenti.

## Funzionalità principali

- Visualizzazione Navi oggi e Navi domani

- Assegnazione dipendenti per fascia oraria

- Gestione navi (creazione, modifica, cancellazione)

- Gestione scadenze patenti e variazioni dei dipendenti

- Persistenza dei dati su file JSON

## Tecnologie utilizzate
 ### Backend
 - ASP.NET Core MVC
 - Entity Framework Core
 - C#
 - SignalR

 ### Database
- SQL Server LocalDB per persistenza dati e connessioni multi-utente

 ### Frontend
  - Vue.js
  - Bootstrap
